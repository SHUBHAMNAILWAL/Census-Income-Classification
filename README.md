# 📊 Census Income Classification

A Machine Learning project that predicts whether an individual's annual income exceeds **$50K** based on demographic and employment-related attributes from the UCI Adult Census Income dataset.

---

## 🚀 Project Overview

The objective of this project is to build a classification model that accurately predicts income categories using census data. The project includes data preprocessing, exploratory data analysis (EDA), feature engineering, model training, and performance evaluation.

---

## ✨ Features

- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Machine Learning Classification
- Model Evaluation & Performance Comparison

---

## 🛠️ Tech Stack

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

---

## 📂 Dataset

- **Dataset:** UCI Adult Census Income Dataset
- **Target Variable:** Income (`<=50K` or `>50K`)

---

## 📈 Project Workflow

1. Data Collection
2. Data Preprocessing
3. Exploratory Data Analysis
4. Feature Engineering
5. Model Training
6. Model Evaluation
7. Prediction

---

## 📊 Model Evaluation

The models were evaluated using:

- Accuracy Score
- Precision
- Recall
- F1-Score
- Confusion Matrix

---

## 🎯 Results

The trained classification model successfully predicts income categories by learning patterns from demographic and employment features, providing reliable performance on unseen data.
